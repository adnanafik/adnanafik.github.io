# adnankhan.me

Personal site for [Adnan Khan](https://adnankhan.me) — Sr. Director of AI Engineering, DevSecOps, and Platform Engineering.

Static HTML + CSS, hosted on GitHub Pages. No build step, no JavaScript framework, no ongoing costs.

## Local preview

Just open `index.html` in a browser. That's it.

## Deploy

Pushing to `main` auto-deploys via GitHub Pages. The `CNAME` file points the custom domain at this repo.
